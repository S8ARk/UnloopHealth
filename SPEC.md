# NutriCore — Project Specification

## Goal
NutriCore is a premium AI-powered nutrition app that provides meal logging, health predictions, and a comprehensive diet management platform.

## Architecture
- **Backend**: FastAPI (Python)
- **Frontend**: React/Vite (TSX)
- **Database**: PostgreSQL (SQLAlchemy) + MongoDB (Meal Logging)
- **Security**: JWT (RSA RS256), Secure HttpOnly Cookies
- **Design System**: Stitch-powered premium UI

## Core Features
1. **Authentication**: Secure login/register with token-based session management.
2. **Meal Tracking**: Log meals with calorie and macro breakdown.
3. **AI Health Prediction**: Advanced dietary risk analysis using meal history and user profiles.
4. **Dashboard**: Premium, responsive dashboard with real-time hydration and nutrition tracking.
